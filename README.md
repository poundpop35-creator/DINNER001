# แผนจัดการหนี้ (Debt Plan)

แอปหน้าเดียว (single-file `index.html`) สำหรับติดตามตารางผ่อนชำระหนี้ 8 เดือน
(ส.ค.–มี.ค.) ของเจ้าหนี้ 5 ราย: **S, G, YourCash, LINE BK, AEON**

## ฟีเจอร์
- การ์ดสรุป: หนี้รวม / จ่ายแล้ว / คงเหลือ / เดือนที่จ่ายแล้ว
- แถบความคืบหน้าโดยรวม และแยกตามเจ้าหนี้แต่ละราย
- ตารางผ่อนรายเดือน พร้อมเครื่องหมาย ✅ ปิด ในเดือนที่ปิดยอดของแต่ละเจ้าหนี้
- ติ๊กเดือนที่จ่ายแล้ว — บันทึกอัตโนมัติในเครื่อง (localStorage)

## วิธีใช้
เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไร
